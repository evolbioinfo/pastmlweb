Install PastML

PastML is composed of two parts: the core C library that provides fast Ancestor Character Reconstruction (ACR) methods without visualisation, and a visualisation library cytopast written in Python 3. The latter wraps the core C library to allow for a seamless use of the whole functionality within Python 3 or from a command line.

PastML with visualisation is also available via docker
