About PASTML

PASTML infers ancestral states on a rooted phylogenetical tree with annotated tips, using maximum likelihood.

The result is then visualised as a zoomable html map.

PASTML code is open source and available on GitHub.

Try it now

You need two files to run PASTML:

  • a rooted tree, in Newick format;
  • an annotation table specifying tip states, in CSV format.
