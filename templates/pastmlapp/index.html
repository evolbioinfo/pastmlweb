About PASTML

PASTML infers ancestral states on a rooted phylogenetical tree with annotated tips, using maximum likelihood.

The tree with reconstructed ancestral states is then visualised as a zoomable html map.

You need two files to run PASTML: one with the rooted reference tree inferred using your preferred method; the second with the tip annotations.

PASTML code is open source and available on GitHub.
